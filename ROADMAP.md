# Roadmap

路线图保持小步迭代，优先提高现有保存流程的稳定性。

## 近期

- 提高知乎、财新和知识星球页面结构变化后的兼容性
- 增加更多脱敏 DOM 测试样本
- 改进首次安装和本地服务状态提示
- 完善 Chrome Web Store 发布材料和隐私说明

## 后续候选

- Windows 和 Linux 的本地服务自启动方案
- 可导入、导出的来源文件夹配置
- 用户明确需要且可以仅依赖当前页面 DOM 的其他中文内容网站

## 不在范围内

- 账号登录和云端同步
- 绕过登录、订阅、付费或网站权限限制
- 调用网站内部 API 批量抓取内容
- 默认上传用户文章内容到第三方服务
- AI 摘要或自动改写正文

路线图不是发布日期承诺。具体优先级会根据真实问题、测试覆盖和维护成本调整。
